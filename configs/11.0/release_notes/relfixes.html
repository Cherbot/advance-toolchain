    • GCC provides:
      • Fix for incorrect code generation for __divkf3.

      • -maltivec and -mvsx GAS options mimic GCC's behaviour.

    • The GNU C Library provides:
      • Fix for CVE-2015-5180: crash with internal QTYPE.
